## 🛠️ Technologies Used

- **HTML5**: Semantic markup for proper structure
- **CSS3**: Advanced styling with grid, flexbox, and animations
- **Responsive Design**: Media queries for mobile optimization

## 📋 Sections

1. **Header/Navigation**: Fixed navigation bar with smooth scrolling
2. **Hero Section**: Eye-catching welcome section with call-to-action buttons
3. **Menu**: Display appetizers and dishes with descriptions and prices
4. **About**: Restaurant story and information about the chef
5. **Reservation Form**: User-friendly form to book a table
6. **Contact**: Contact details and location information
7. **Footer**: Quick links, opening hours, and social media

## 🎨 Design Features

- **Color Scheme**: Professional red (#e74c3c) and dark blue (#2c3e50)
- **Animations**: Fade-in and slide-in effects for visual appeal
- **Typography**: Clean, readable fonts with proper hierarchy
- **Interactive Elements**: Hover effects on buttons, menu items, and links

## 📱 Responsive Breakpoints

- **Desktop**: Full layout with 2-column grid
- **Tablet** (max-width: 768px): Single column layout
- **Mobile** (max-width: 480px): Optimized for small screens

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Rudra-729/Bella-Vita.git
```

2. Navigate to the project directory:
```bash
cd Bella-Vita
```

3. Open `index.html` in your web browser to view the website

## 📝 Form Features

- **Reservation Form**: Collect customer details for table booking
- **Form Validation**: Required fields for better data quality
- **Date & Time Selection**: Easy selection of preferred reservation time
- **Guest Count**: Options for 1 to 5+ guests

## 🔧 Customization

To customize the website:

1. **Colors**: Edit CSS variables in `styles.css` (`:root` section)
2. **Content**: Update text in `index.html`
3. **Images**: Replace image paths with your own
4. **Menu Items**: Add or remove menu items as needed
5. **Contact Info**: Update address, phone, and email
